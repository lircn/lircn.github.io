上次建站是在5年前，心血来潮地搞了个基本没有blog的Blog，摸索中把前后端（Bootstrap + Nginx + Tornado）撸了一遍，顺便买买CVM，域名和带宽，好给前东家贡献一点软妹币， 提前体验到全栈的残酷。
充钱你就会变强.jpg

如今回顾，角落的站点许久未更新，表面的设计和背后的代码都诉说着当年的Trash稚嫩。
是时候重构了，趁着再次心血来潮的冲动，往前多走几步。

清空所有旧资源和代码，全新的前后端，from here，from now。

来自远方，
来自黄昏和清晨，
来自十二重高天的好风轻扬，
飘来生命气息的吹拂：
吹在我身上。
快，趁生命气息逗留，
盘桓未去，
拉住我的手，
快告诉我你的心声。
